# TC_007: ファイル読み込み成功テスト

## テストケース情報

| **テスト分類** | 単体テスト |

| **対象メソッド** | `readFile(path)` |
| **優先度** | 高 |

## テスト内容

### 事前条件（試験規格）
- 指定パスにtxtファイルが存在する

### 判定条件（試験合格値）
- ファイル内容が文字列として返却される

## 検証項目

1. テスト用ファイルを一時的に作成
2. `readFile(path)`の返り値が文字列
3. 返り値がファイルの実際の内容と一致

## テストコード例

```javascript
test('ファイル内容が文字列として返却される', () => {
  const testContent = 'これはテストファイルの内容です。';
  fs.writeFileSync(testFilePath, testContent, 'utf8');

  const fileReader = new FileReader();
  const result = fileReader.readFile(testFilePath);

  expect(typeof result).toBe('string');
  expect(result).toBe(testContent);
});

test('日本語を含むファイルが正しく読み込める', () => {
  const testContent = 'システム設計に関する文書です。\n改行も含みます。';
  fs.writeFileSync(testFilePath, testContent, 'utf8');

  const fileReader = new FileReader();
  const result = fileReader.readFile(testFilePath);

  expect(result).toBe(testContent);
});
```

## 実装コード例

```javascript
const fs = require('fs');

class FileReader {
  readFile(path) {
    const content = fs.readFileSync(path, 'utf8');
    return content;
  }
}
```
