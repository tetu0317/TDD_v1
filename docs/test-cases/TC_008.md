# TC_008: ファイル読み込み失敗テスト

## テストケース情報

| 項目 | 内容 |
|------|------|
| **テストケースID** | TC_008 |
| **テスト分類** | 単体テスト |
| **対象クラス** | FileReader |
| **対象メソッド** | `readFile(path)` |
| **優先度** | 高 |

## テスト内容

### 事前条件（試験規格）
- 指定パスにファイルが存在しない

### 判定条件（試験合格値）
- エラーがthrowされ、エラーメッセージに"File not found"が含まれる

## 検証項目

1. 存在しないパスを指定
2. `readFile("/non/existent/path.txt")`がエラーをthrow
3. エラーメッセージに"File not found"または"ENOENT"が含まれる

## テストコード例

```javascript
test('存在しないファイルを読み込むとエラーがthrowされる', () => {
  const fileReader = new FileReader();
  const nonExistentPath = '/tmp/non-existent-file.txt';

  expect(() => {
    fileReader.readFile(nonExistentPath);
  }).toThrow();
});

test('エラーメッセージに"File not found"または"ENOENT"が含まれる', () => {
  const fileReader = new FileReader();
  const nonExistentPath = '/tmp/non-existent-file.txt';

  expect(() => {
    fileReader.readFile(nonExistentPath);
  }).toThrow(/File not found|ENOENT/);
});
```

## 実装コード例

```javascript
const fs = require('fs');

class FileReader {
  readFile(path) {
    try {
      const content = fs.readFileSync(path, 'utf8');
      return content;
    } catch (error) {
      if (error.code === 'ENOENT') {
        throw new Error(`File not found: ${path}`);
      }
      throw error;
    }
  }
}
```
