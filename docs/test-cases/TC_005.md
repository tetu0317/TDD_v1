| **テスト分類** | 単体テスト |

| **対象メソッド** | `search(keyword)` |
| **優先度** | 中 |

## テスト内容

### 事前条件（試験規格）
- ドキュメントが登録済み

### 判定条件（試験合格値）
- 存在しないキーワードで検索時、空配列が返却される

## 検証項目

1. `search("存在しないキーワード")`の結果が空配列`[]`
2. 結果の長さが0

## テストコード例

```javascript
test('存在しないキーワードで検索時、空配列が返却される', () => {
  const searcher = new DocumentSearcher();
  searcher.addDocument('設計書.txt', '/documents/design.txt', 'システム設計に関する文書です。');

  const results = searcher.search('存在しないキーワード');

  expect(Array.isArray(results)).toBe(true);
  expect(results).toHaveLength(0);
  expect(results).toEqual([]);
});
```
